# Домашние задания по курсу "Основы информационного поиска"

Репозиторий содержит практические домашние задания, выполненные в рамках профильного курса на программе ПМИ ФКН ВШЭ. Проекты посвящены реализации ключевых структур данных внешней памяти, алгоритмов индексации текстовых и многомерных данных, а также систем ранжирования.

## Стек технологий и базовые концепты
* Языки и инструменты: Python, Makefile
* Структуры данных: LSM-дерево, обратный индекс, Bit-Sliced Indexing (BSI), позиционный индекс, k-d дерево, Roaring Bitmap
* Алгоритмы ранжирования: TF-IDF

---

## Структура репозитория

Каждая директория соответствует определенной теме курса и содержит программную реализацию подсистем поискового движка:

*   **hw1 - Деревья со слиянием (LSM-Tree)**
    *   Реализация архитектуры хранения данных во внешней памяти. Проектирование компонентов MemTable и неизменяемых SSTable, механизмов слияния и уплотнения (Compaction) данных.
*   **hw2 - Обратный индекс (Inverted Index)**
    *   Построение классического инвертированного индекса для обеспечения эффективного булева поиска по текстовым корпусам документов.
*   **hw4 - Даты и BSI (Bit-Sliced Indexing)**
    *   Реализация побитового индексирования (Bit-Sliced Indexing) для оптимизации и ускорения аналитических запросов по диапазонам дат.
*   **hw5 - Позиционный индекс (Positional Index)**
    *   Расширение структуры обратного индекса с сохранением позиций токенов в документах для поддержки точного фразового поиска.
*   **hw6 - Ранжирование текстовых документов**
    *   Реализация математических моделей оценки релевантности текстовых документов.
*   **hw7 - Многомерный поиск и пространственные индексы**
    *   Реализация k-d дерева для выполнения range-запросов и поиска ближайших соседей.

---

## Дополнительно изученные темы курса
Помимо выполнения базовых домашних заданий, в рамках теоретической программы курса были подробно изучены и разобраны следующие концепты:
* **Поиск по маскам (Wildcard-поиск) и обработка опечаток:** Построение словарей из перевернутых слов для суффиксального поиска, выполнение Range-запросов по диапазонам токенов, а также использование n-grams (триграмм) для поиска произвольных шаблонов (масок типа `prefix*suffix`) с оптимизацией через Блум-фильтры и Point lookups.
* **Сжатые полнотекстовые индексы:** Преобразование Барроуза-Уилера (BWT), wavelet tree, LF-свойство (Last-First mapping) и FM-index для эффективного поиска подстрок в сжатых массивах данных.
* **Семантический поиск:** Построение векторных индексов по эмбеддингам текстов (LSA/LSI, сингулярное разложение SVD, Randomised PCA) и использование мер косинусного сходства для извлечения документов по смыслу.
